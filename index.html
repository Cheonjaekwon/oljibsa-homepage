잔디는 깔끔하게,
전원생활은 편안하게.

시간이 부족하거나 직접 관리하기 힘든 잔디와 외부 공간을 올집사가 관리합니다.
현장 사진 3~5장을 보내주시면 필요한 작업과 예상 금액을 먼저 안내드립니다.

서비스 안내

필요한 관리만 선택하실 수 있습니다.

1

기본 잔디관리

잔디깎기 + 가장자리 예초 + 작업 후 한곳 정리

평당 1,000원최소 작업금액 30,000원
2

제초관리

기본 잔디관리 + 필요한 구역의 잡초관리

50평 기준 70,000원부터기본관리 + 선택성 제초·표준 소모재
3

생육관리

제초관리 + 잔디 비료 시비

50평 기준 85,000원부터제초관리 + 비료·표준 소모재
4

부분복토

꺼진 곳이나 맨땅을 중심으로 필요한 부분만 복토

50평 기준 125,000원부터부분복토 인건비 포함 · 복토재 실비 별도
작업 후 나온 잔디와 잡초는 고객 부지 내 지정 장소에 모아드립니다.
폐기물 수거·외부 반출은 현재 제공하지 않습니다.

이용 과정

간단한 4단계로 진행됩니다.

1 사진 보내기

마당 전체와 상태를
3~5장 보내주세요.

2 견적 안내

사진 확인 후 작업 가능 여부와
예상 금액 안내

3 방문 작업

예약한 일요일 오전,
방문하여 작업 진행

4 작업 결과 보고

작업 전후 사진과 내용을
전달해 드립니다.

사진 요청 기준 (3~5장)

  • 마당 전체 전경
  • 좌측과 우측
  • 잔디·잡초 상태 근접 사진
  • 진입로 또는 장애물
  • 폐기물이나 적치물이 있다면 해당 사진

여러 작업을 함께 요청하시면

잔디깎기·잡초관리·마당정리를 같은 날 함께 진행해 드립니다. 출장과 준비 시간이 줄어 더 합리적인 묶음 견적으로 안내드립니다.

견적 기준

현장 난이도, 경사, 돌·장애물, 잔디 높이, 진입 여건에 따라 사진 견적에서 금액이 조정될 수 있습니다.
